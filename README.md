# Credit-Card-Analysis
This project contains two interactive Power BI dashboards built to analyze credit card customer behavior and transaction performance. The dashboards provide insights into revenue, transactions, customer demographics, and spending patterns.
# Dashboard 1: Credit Card Customer Report
## Key KPIs
* Total Revenue
* Total Interest Earned
* Total Customer Income
* Customer Satisfaction Score
## Insights
* Revenue vs Gender trend
* Customer distribution by Age Group
* Income Group analysis
* Marital Status breakdown
* Education Level distribution
* Top 5 States by Revenue
* Revenue by Customer Job
# Dashboard 2: Credit Card Transaction Report
## Key KPIs
* Total Revenue
* Total Interest
* Transaction Amount
* Transaction Count
## Insights
* Quarterly Revenue and Transaction Trends
* Revenue by Card Category
* Revenue by Expenditure Type
* Revenue by Customer Job
* Revenue by Education Level
* Revenue by Chip Usage
* Customer Acquisition Cost analysis
# Tools Used
* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization
# Dataset
* customer.csv – Customer demographic information
* credit_card.csv – Credit card transaction data
# Key Features
* Interactive slicers and filters
* KPI cards for business metrics
* Dynamic charts and visuals
* Clean and modern dashboard design
# Outcome
The dashboards help analyze customer behavior, transaction trends, and revenue performance to support data-driven decision making.
